# Enquiry Management System

A full-stack web application that allows users to submit enquiries and store them in a database. The system helps manage and organize user enquiries efficiently.

---

## Features

* Submit enquiries through a web form
* Store enquiries in a database
* Retrieve and manage enquiries

---

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
